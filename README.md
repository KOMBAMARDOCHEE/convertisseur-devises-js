#  Convertisseur de Devises — JavaScript

##  Description
Application de conversion de devises utilisant des taux simulés pour démontrer la logique métier et la manipulation de données en JavaScript.

##  Objectifs techniques
- Manipulation des données
- Calculs dynamiques
- Interaction avec des formulaires HTML

##  Technologies utilisées
- HTML5
- CSS3
- JavaScript (ES6)

##  Fonctionnalités
- Conversion entre USD, EUR et XOF
- Calcul automatique du montant converti
- Interface claire et accessible

##  Lancer le projet
1. Ouvrir le fichier `index.html` dans un navigateur
2. Entrer un montant et sélectionner les devises

##  Auteur
Projet démontrant la logique métier côté client en JavaScript.
